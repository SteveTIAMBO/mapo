# MAPO — ERP Scolaire par EDUFREM

MAPO est un système de gestion scolaire (ERP) conçu spécifiquement pour les écoles africaines.

## Caractéristiques principales
- **Offline-first** : fonctionne sans connexion internet, synchronisation automatique au retour du réseau
- **PWA installable** : s'installe comme une application native sur mobile et tablette
- **Import Excel** : démarrage rapide via import de fichiers Excel modèles
- **Multi-rôles** : Admin, Directeur, Enseignant, Parent, Élève

## Stack technique
- Vue.js 3 + Vite
- Firebase (Firestore + Auth + Storage)
- Tailwind CSS
- Pinia (state management)

## Modules
- 👨‍🎓 Gestion des élèves
- 🏫 Classes & groupes
- 📝 Notes & bulletins
- ✅ Présences
- 📅 Emploi du temps
- ⚖️ Discipline
- 💰 Facturation
- 📊 Rapports PDF
- 📥 Import Excel
